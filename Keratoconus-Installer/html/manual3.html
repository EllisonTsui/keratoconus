Presentation of results

The application offers several ways to present the results of the analysis to the user:

  • animation of the application's icon in the task bar - the colour gives an indication of the category.
  • showing a small pop-up message (“balloon”) next to the task bar icon.
  • a pop-up window which shows the patient ID, any warnings reported by the Pentacam software, and the full results of the analysis.

It is possible to select which of these are used via the “Results display” sub-menu of the User preferences … menu. Even if the pop-up window is not enabled here it is always available by double-clicking on the task bar icon. It can be dismissed by clicking anywhere inside the window.

The pop-up window is divided into four main zones:

Identity zone
This contains the patient's name, date of birth and ID and the details of the examination (eye, date, and time).
Warning zone
If present this contains a warning (yellow background) or error (red background) reported by the Pentacam software. In the case of an error the data is considered to be unusable and the remaining zones will not be shown.
Headline zone
This contains a short description of the classification (one or two words), shown in block capitals. If the classification is No indication or Thick Cornea then the detail zone will not be shown.
Detail zone
Here the most likely classifications are listed in smaller print. The “?” button provides access to a graphical presentation of the probabilities estimates provided by the model.
