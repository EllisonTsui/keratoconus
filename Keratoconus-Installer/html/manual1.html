About the Keratoconus Assistant

Notice

The Keratoconus Assistant software package is intended as a decision support system, providing physicians with an automated screening tool for keratoconus. This software is not intended to replace the comprehensive anamnesis, ophthalmic examination or appropriate diagnostic measurements deemed the standard protocol in keratoconus assessment. Instead, it is to be used in conjunction with this standard protocol, and provide an objective point of view based on machine learning analysis. The results provided by the Keratoconus Assistant are therefore not to be considered as a medical instruction by the developers. The responsibility for any medical decisions based on this software’s results remains solely with the physician using it, who should come to his own, independent conclusions regarding the proper treatment of their patient, taking all available medical information into account.

By using Keratoconus Assistant the user accepts the full responsibility for the interpretation of the results of the software in their proper clinical context, taking all available information into account. Furthermore, by using Keratoconus Assistant the user pledges to refrain from making any claims against the developers related to the use of the software.

General principles

The Keratoconus Assistant is an expert system developed at the University Hospital (Universitair Ziekenhuis) of Antwerp to assist ophthalmic practitioners in detecting cases of Keratoconus. It functions in association with the Oculus Pentacam software; each time that a corneal tomography scan is performed or a previous scan is recalled, the data provided by the Pentacam software is fed into a neural network which has been trained to recognise possible cases of Keratoconus. The result of this analysis is then displayed in a pop-up window.
