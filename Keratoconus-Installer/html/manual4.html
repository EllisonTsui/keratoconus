Patient record files

In order to provide a permanent record, the full output of each analysis is append to the file C:\Pentacam\Keratoconus-Assistant.CSV.

If a target directory has been selected using the “Patient records” sub-menu of the User preferences … menu, a JPEG file containing this information will be created in this directory for each patient examination. The file name has the same structure (LastName_FirstName_Eye_Date_Time_KeratoconusAssistant.JPG) as the graphical files created by the Pentacam software, making it easy to correlate the files.
